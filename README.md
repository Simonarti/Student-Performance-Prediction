# 🎓 Student Performance Prediction (ML + Streamlit)

## 📌 Overview

This project is an end-to-end Machine Learning application that predicts student performance based on various input features such as study habits, attendance, and academic history.

The project involves data preprocessing, model building, evaluation, and deployment using an interactive web application.

---

## 🚀 Features

* Data preprocessing and cleaning
* Feature selection and transformation
* Model training using:

  * Logistic Regression
  * Random Forest Classifier
* Model comparison and evaluation
* Interactive web application for real-time predictions

---

## 🧠 Machine Learning Models Used

* **Logistic Regression** (Best Performing Model ✅)
* Random Forest Classifier

Logistic Regression achieved better performance compared to Random Forest based on evaluation metrics.

```

```
